Questionnaire: 

1. What are javascript functions?
A SET OF INSTRUCTIONS OR STATEMENT THAT PERFORMS A TASK OR CALCULATES A VALUE.  TAKES INPUT AND RETURNS SOME OUTPUT.

2. Explain call-back functions in your own words.
FUNCTIONS ARE OBJECTS.  A CALLBACK FUNCTION ACCEPTS ANOTHER FUNCTION AS AN ARGUMENT.  INVOKING THAT FUNCTION WHEN IT COMPLETES.

3. Why are javascript functions first-class citizens? (in your own words, please)
IT IS IMPORTANT THAT FUNCTIONS BE TREATED LIKE ANY OTHER FIRST-CLASS OBJECT.  THIS ALLOWS THEM TO BE PASSED AROUND, STORED IN VARIABLES, RETURNED TO OTHER FUNCTIONS, AND HOLD THEIR OWN PROPERTIES.

4. Explain javascript Closures in your own words.
CLOSURE IS MOST OFTEN USED FOR FUNCTIONS RETURNING FUNCTIONS FOR FUTURE USE.  WHEN YOU HAVE A FUNCTION DEFINED INSIDE AN INNER FUNCTION / THAT FUNCTION HAS ACCESS TO THE VARIABLES AND SCOPE OF THE OUTER FUNCTION / EVEN IF THAT FUNCTION FINISHES EXECUTING AND THOSE VARIABLES ARE NO LONGER ACCESSIBLE.

5. When/Why do you use the .then() method of a promise?
.THEN() IS DEFINED AS A PROMISE THAT IS USED TO DEALING WITH ASYNCHRONOUS TASKS.  ESPECIALLY WHEN BEING USED FOR API CALLS AS TIME VARIES FROM CALL TO CALL.

6. When/Why do you use the .catch() method of a promise?
.CATCH() IS THE METHOD USE WHEN YOUR PROMISE HAS BEEN REJECTED.  IT IS EXECUTED IMMEDIATELY AFTER A PROMISE REJECTS METHOD IS CALLED.


7. What is the spread operator?
TAKES AN ARRAY AND EXPANDS IT TO INDIVIDUAL ITEMS

8. What is the rest operator?
ALLOWS AN INDEFINITE NUMBER OF ARGUMENTS AS AN ARRAY.

9. What is the difference between the rest operator and the spread operator?
THE REST OPERATOR PUTS THE REST OF SOME SPECIFIC USER-SUPPLIED VALUES INTO A JAVASCRIPT ARRAY.

THE SPREAD SYNTAX EXPANDS ITERABLES INTO INDIVIDUAL ELEMENTS.

10. What is Object and Array Destructuring in javascript?
IS A WAY TO UNPACK VALUES FROM ARRAYS, OR PROPERTIES FORM OBJECTS INTO DISTINCT VARIABLES.
